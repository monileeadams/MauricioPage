# **App Name**: Terra Vision

## Core Features:

- Hero Slider: Rotating showcase of cinematic and natural imagery on the homepage.
- Featured Quote Display: Display a prominent, impactful quote related to environmental justice.
- Portfolio Grid: Showcase featured projects and productions with an image grid layout.
- Service Listing: Highlight available services in a clear, icon-based format.
- Gallery Organization: Categorize gallery images for easy browsing by nature, film projects, etc.
- Blog Post Display: Display the latest blog articles.
- Contact Form Submission: Enable users to send direct messages.

## Style Guidelines:

- Primary color: White (#FFFFFF) for the main background to ensure content clarity.
- Secondary color: Very light gray (#F5F5F5) used for alternating sections to create visual separation.
- Accent color: Teal (#26a69a) for buttons, links, and interactive elements, conveying a sense of environmental awareness.
- Headline Font: 'Playfair' a serif, for titles, to evoke elegance and readability. Note: currently only Google Fonts are supported.
- Body Font: 'Montserrat' a sans-serif, providing a clean and modern feel for body text. Note: currently only Google Fonts are supported.
- Use a consistent style of outline icons to represent services and social media links.
- Implement a responsive design that adapts seamlessly to different screen sizes.
- Incorporate subtle animations and transitions to enhance user engagement and provide visual feedback.
